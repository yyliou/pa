為關係建模

一個值得建模的問題

棒球隊的得分能由打擊數據解釋多少？2000–2008 年的球隊打擊資料，依課程慣例載入：

temp <- tempfile(fileext = ".rda")
download.file("https://raw.githubusercontent.com/cran/nutshell/master/data/team.batting.00to08.rda", temp, mode = "wb")
load(temp)

線性模型假設

\[\text{runs} = c_0 + c_1 x_1 + \cdots + c_n x_n + \varepsilon,\]

lm() 以最小平方法估計係數。

配適模型

runs.mdl <- lm(runs ~ singles + doubles + triples + homeruns +
                 walks + hitbypitch + sacrificeflies +
                 stolenbases + caughtstealing,
               data = team.batting.00to08)

穩健與抗性迴歸

離群值能把最小平方法拖去任何地方。穩健方法會調降它們的權重：

library(MASS)
runs.rlm <- rlm(runs ~ singles + doubles + triples + homeruns + walks,
                data = team.batting.00to08)
round(coef(runs.rlm), 3)
(Intercept)     singles     doubles     triples    homeruns       walks 
   -501.236       0.593       0.739       1.211       1.530       0.318
  • rlm()——M 估計（MASS）；lqs()——least trimmed squares 等抗性配適。
  • 穩健配適與普通配適差很大時，先調查高影響觀測值，再決定相信誰。

子集選擇與收縮

預測變數多且相關時，三種策略控制變異：

  • 子集選擇：step()，或 leaps 套件的窮舉搜尋。
  • 脊迴歸：保留所有項、收縮係數（\(L_2\) 懲罰）——MASS 的 lm.ridge()。
  • Lasso：又收縮又選擇（\(L_1\) 懲罰）——glmnet 套件能有效率地算出整條正則化路徑。
library(glmnet)
fit <- glmnet(x = as.matrix(predictors), y = runs, alpha = 1)  # lasso
cv  <- cv.glmnet(as.matrix(predictors), runs)   # 以交叉驗證挑 lambda

主成分迴歸與偏最小平方迴歸（pls 套件）是投影法的表親。

廣義線性模型

glm() 透過連結函數與誤差家族擴充線性模型——計數、比率、機率共用一個介面：

counts <- data.frame(events = c(2, 1, 4, 7, 9, 14),
                     exposure = 1:6)
fit.pois <- glm(events ~ exposure, family = poisson, data = counts)
coef(summary(fit.pois))
               Estimate Std. Error     z value     Pr(>|z|)
(Intercept) -0.04133056  0.5588550 -0.07395579 9.410456e-01
exposure     0.45144078  0.1142353  3.95184880 7.754973e-05
  • gaussian 重現 lm()；poisson 建模計數；binomial 是邏輯斯迴歸（第二十一章）。
  • 係數活在連結尺度上——Poisson／logit 係數要取指數才好詮釋。

非線性與平滑模型

關係真的彎曲時：

fit.lo <- loess(dist ~ speed, data = cars)
plot(cars, pch = 19)
lines(seq(4, 25), predict(fit.lo, data.frame(speed = seq(4, 25))),
      col = "red", lwd = 2)
  • nls() 配適參數式非線性曲線（公式與起始值由你提供）。
  • loess() 局部多項式；smooth.spline() 懲罰式樣條；mgcv::gam() 完整的廣義可加模型。

存活模型

含設限的事件時間資料用 survival 套件：

library(survival)
fit.km <- survfit(Surv(time, status) ~ x, data = aml)
plot(fit.km, lty = 1:2, xlab = "Weeks", ylab = "Survival")
legend("topright", c("Maintained", "Nonmaintained"), lty = 1:2)
  • survfit()——Kaplan–Meier 曲線；survdiff()——log-rank 檢定；coxph()——Cox 比例風險迴歸。
  • Surv(time, status) 物件編碼設限；忽略設限會讓一切結果偏誤。

一份建模食譜

  1. 先畫圖（第十三～十五章）；猜函數形式。
  2. 用 lm()/glm() 配適最簡單的合理模型。
  3. 診斷：plot(model)、殘差、影響度。
  4. 精煉：轉換、增刪項，或正則化。
  5. 在保留資料上驗證（第二十二章的紀律）。
  6. 用 confint() 報告效果量，別只報星號 (Hastie et al. 2009)。
