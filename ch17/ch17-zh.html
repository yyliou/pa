d-p-q-r 系統

四個字母，所有分布

對每個分布 xxx，R 提供四個函式：

字首 函式 回答的問題
d 密度／質量 值 \(x\) 出現的可能性多大？
p 累積分布 \(P(X \le q)\)？
q 分位數 哪個值以下累積了機率 \(p\)？
r 隨機生成 給我 \(n\) 個隨機值 
dnorm(0)              # 標準常態在 0 的密度
[1] 0.3989423
pnorm(1.96)           # P(Z <= 1.96)
[1] 0.9750021
qnorm(0.975)          # 把著名的 1.96 找回來
[1] 1.959964
set.seed(1); rnorm(3) # 三個常態隨機數
[1] -0.6264538  0.1836433 -0.8356286

常態分布

\[f(x) = \frac{1}{\sigma\sqrt{2\pi}}\, e^{-\frac{(x-\mu)^2}{2\sigma^2}}\]

x <- seq(-4, 4, length.out = 200)
plot(x, dnorm(x), type = "l", lwd = 2, ylab = "density")
lines(x, dnorm(x, sd = 1.5), lty = 2)
legend("topright", c("sd = 1", "sd = 1.5"), lty = 1:2)

mean= 平移、sd= 伸縮；預設值就是標準常態。

看見 p 與 q

著色面積就是 pnorm()；qnorm() 是反向查表：

x <- seq(-4, 4, length.out = 200)
plot(x, dnorm(x), type = "l", lwd = 2, ylab = "density")
xs <- seq(-4, qnorm(0.975), length.out = 120)
polygon(c(xs, rev(xs)), c(dnorm(xs), rep(0, 120)),
        col = "lightsteelblue", border = NA)
abline(v = qnorm(0.975), lty = 2)
text(2.6, 0.1, "97.5%")

連續分布家族

同一套四字母模式涵蓋所有經典連續家族：

punif(0.5)                  # [0,1] 上的均勻分布
[1] 0.5
pexp(1, rate = 1)           # 指數分布
[1] 0.6321206
pt(2.086, df = 20)          # Student's t
[1] 0.9750018
pchisq(3.84, df = 1)        # 卡方
[1] 0.9499565
pf(4.35, df1 = 2, df2 = 20) # F
[1] 0.9729927
  • t、chisq、f 是第十八章檢定的主力——它們的 q 函式就能生成你以前在附錄查的臨界值表。
  • 還有：lnorm、gamma、beta、weibull、cauchy 等等。

隨機數與可重現性

set.seed(2026)
rbinom(5, size = 10, prob = 0.3)
[1] 4 3 1 2 3
sample(c("H", "T"), 5, replace = TRUE)
[1] "H" "H" "T" "T" "T"

Tip

作業與報告永遠先 set.seed()。 偽隨機序列從種子出發完全可重播，閱卷者——以及未來的你——才能一字不差地重現每個模擬。

模擬作為工具

理論算不動時，就模擬。擲五顆骰子，最大值是 6 的機率是多少？

set.seed(7)
sims <- replicate(10000, max(sample(1:6, 5, replace = TRUE)))
mean(sims == 6)
[1] 0.6019
1 - (5/6)^5          # 精確答案，拿來對照
[1] 0.5981224

replicate() 加上 r 函式，把機率謎題變成十秒鐘的實驗——拔靴法（第十六章）與檢定力分析（第十九章）用的正是同一招。
