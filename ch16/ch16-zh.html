從資料擠出第一批數字

摘要統計量

描述統計的基本詞彙，每個都只差一個函式：

temp <- tempfile(fileext = ".rda")
download.file("https://raw.githubusercontent.com/cran/nutshell/master/data/dow30.rda", temp, mode = "wb")
load(temp)
x <- dow30$Open
mean(x); median(x)
[1] 36.24574
[1] 30.155
var(x); sd(x)
[1] 495.1713
[1] 22.25245
range(x); quantile(x, c(0.25, 0.5, 0.75))
[1]   0.99 122.45
   25%    50%    75% 
19.655 30.155 51.680
  • mean(x, trim = 0.1) 抵抗離群值；這裡每個函式都接受 na.rm=TRUE。
  • fivenum() 給出 Tukey 五數摘要；IQR() 是四分位距。

  • cor() 預設 Pearson；method = "spearman" 或 "kendall" 改用等級——對離群值與非線性都更穩健。
  • 相關矩陣是迴歸（第二十章）前的標準偵察。

Warning

相關不是因果。 它也看不見非線性結構——散佈圖永遠要一起看。

主成分分析

PCA 把彼此相關的變數重新表達成依解釋變異量排序的正交主成分：

pca <- prcomp(USArrests, scale. = TRUE)
summary(pca)
Importance of components:
                          PC1    PC2     PC3     PC4
Standard deviation     1.5749 0.9949 0.59713 0.41645
Proportion of Variance 0.6201 0.2474 0.08914 0.04336
Cumulative Proportion  0.6201 0.8675 0.95664 1.00000
biplot(pca, cex = 0.6)
  • scale.=TRUE 先標準化——單位不同時必須做。
  • 前兩個主成分常已捕捉大部分結構：替繪圖與建模內建了降維。

因素分析

拔靴法重抽樣

當統計量的不確定性沒有現成公式時，就重抽：放回地抽取觀測值、重算、重複：

library(boot)
med.fun <- function(d, i) median(d[i])
set.seed(1)
b <- boot(dow30$Open, med.fun, R = 1000)
b

ORDINARY NONPARAMETRIC BOOTSTRAP


Call:
boot(data = dow30$Open, statistic = med.fun, R = 1000)


Bootstrap Statistics :
    original  bias    std. error
t1*   30.155  0.0087   0.2879809
boot.ci(b, type = "perc")$percent
     conf                          
[1,] 0.95 25.03 975.98 29.63 30.735

一千次重算的中位數的散布，就是抽樣分布的估計——完全不需要常態假設。

一套工作流程

  1. str() 與 summary()——型別、範圍、缺漏。
  2. 畫圖——直方圖與散佈圖（第十三～十五章）。
  3. 相關——什麼跟什麼一起動？
  4. 降維——變數很多時用 PCA／因素分析。
  5. 量化不確定性——有疑慮就拔靴。

之後——也只有在之後——才輪到第十八章的正式檢定與第二十、二十一章的模型。
