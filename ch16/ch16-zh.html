相關與共變異

三種相關係數

相關回答「x 上升時 y 是否跟著上升、咬得多緊？」，以 −1…1 度量線性相依。三種統計量：

  • Pearson——Excel 的 CORREL；植根於常態理論，最適合近似常態的資料；
  • Spearman——無母數、不做分布假設（對等級做相關）；
  • Kendall 的 tau——比較成對排序的一致與不一致對數。
cor(x, y = NULL, use = "everything",
    method = c("pearson", "kendall", "spearman"))

x、y 可以是兩個向量，或一個資料框／矩陣（→ 相關矩陣）。use 處理 NA："all.obs"（報錯）、"everything"（結果為 NA）、"complete.obs"／"na.or.complete"（丟不完整列）、"pairwise.complete.obs"（逐對丟）。

範例：孕期增重與出生體重

母親增重與寶寶體重相關嗎？先清理（有效值、單胞胎、妊娠 > 35 週），再看圖——十萬多點得用 smoothScatter、不能用 plot：

temp <- tempfile(fileext = ".rda")
download.file("https://raw.githubusercontent.com/cran/nutshell/master/data/births2006.smpl.rda",
              temp, mode = "wb")
load(temp)
births2006.cln <- births2006.smpl[
  !is.na(births2006.smpl$WTGAIN) & !is.na(births2006.smpl$DBWT) &
  births2006.smpl$DPLURAL == "1 Single" & births2006.smpl$ESTGEST>35,]
smoothScatter(births2006.cln$WTGAIN, births2006.cln$DBWT)
cor(births2006.cln$WTGAIN, births2006.cln$DBWT)
[1] 0.1751866
cor(births2006.cln$WTGAIN, births2006.cln$DBWT, method="spearman")
[1] 0.1776192

兩種度量都顯示中等偏弱的相關——那團雲確實有點傾角，但只是一點。（書中用的是 320 萬筆的完整檔；我們的 10% 樣本給出幾乎相同的值。）

共變異

共變異（covariance）就是 Pearson 公式的分子——引數與 cor 相同：

cov(births2006.cln$WTGAIN, births2006.cln$DBWT)
[1] 1176.469

親戚：cov2cor 把共變異矩陣轉成相關矩陣；cov.wt(x, wt=, cor=, center=, method=c("unbiased","ML")) 計算加權共變異。

主成分分析

prcomp 與 princomp

PCA 把（可能相關的）變數改寫成互不相關的成分。兩個實作：prcomp（建議使用；公式法 prcomp(~..., data=, subset=, na.action=)，預設法有 retx、center=TRUE、scale.、tol）與 S-PLUS 相容的 princomp（對相關／共變異矩陣做特徵分解）——書中示範的是後者。

資料：2000–2008 各隊打擊統計，從 Baseball Databank 以 SQL 取出（縮寫改成易讀的名稱），打包為 team.batting.00to08：

temp <- tempfile(fileext = ".rda")
download.file("https://raw.githubusercontent.com/cran/nutshell/master/data/team.batting.00to08.rda",
              temp, mode = "wb")
load(temp)
batting.pca <- princomp(~singles+doubles+triples+homeruns+
                          walks+hitbypitch+sacrificeflies+
                          stolenbases+caughtstealing,

因素分析

factanal

有些量觀測得到（測驗分數）、有些不行（智力）。因素分析從觀測變數推測隱藏因素——stats 的 factanal：

factanal(x, factors, data = NULL, covmat = NULL, n.obs = NA,
         subset, na.action, start = NULL,
         scores = c("none", "regression", "Bartlett"),
         rotation = "varimax", control = NULL, ...)
引數 說明
x／data 公式或數值矩陣（公式時加資料框）
factors 要配適幾個因素
covmat／n.obs 也可給共變異矩陣（或 cov.wt 輸出）＋觀測數
subset／na.action／start 老面孔
scores "none"、"regression"（Thompson）、"Bartlett"（加權最小平方）
rotation 因素旋轉函式，預設 "varimax"

拔靴重抽樣

想法，以及 boot

你的統計量對少數離群值敏感嗎？它的抽樣範圍多大？拔靴法（bootstrap）對任意統計量都能回答：反覆放回抽樣、重算統計量、觀察其散布——正式地說，是估計一個估計量的偏誤。

library(boot)
boot(data, statistic, R, sim="ordinary", stype="i",
     strata=rep(1,n), L=NULL, m=0, weights=NULL,
     ran.gen=function(d, p) d, mle=NULL, simple=FALSE, ...)

重點引數：data；statistic——一個（資料, 選擇）的函式，第二個引數的意義由 stype 決定（"i" 索引、"f" 次數、"w" 權重）；R——重抽次數；sim——"ordinary"、"parametric"、"balanced"、"permutation"、"antithetic"；多樣本問題用 strata；參數化模擬用 ran.gen／mle；simple=TRUE 以時間換記憶體。

範例：中位數有多偏？

媒體報導房價中位數。對 2008 年 6 月舊金山成交價拔靴：

temp <- tempfile(fileext = ".rda")
download.file("https://raw.githubusercontent.com/cran/nutshell/master/data/sanfrancisco.home.sales.rda",
              temp, mode = "wb")
load(temp)
home.sale.prices.june2008 <- subset(sanfrancisco.home.sales,
  format(date, "%Y-%m")=="2008-06")$price
library(boot)
b <- boot(data=home.sale.prices.june2008,
          statistic=function(d, i) { median(d[i]) },
          R=1000)
b
